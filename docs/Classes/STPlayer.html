STPlayer

This class represents an audio player which allow you to play tracks or playlist.

Info:

This object is implementing an AVPlayer to manage all audio within the SDK.

Usage Example:

 private func playTrack() {
     let track = STTrack()
     STPlayer.shared?.play(track)
 }

 private func playPlaylist() {
     let playlist = STPlaylist()
     STPlayer.shared?.play(playlist)
 }
